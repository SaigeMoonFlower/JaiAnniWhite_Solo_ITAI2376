# Reflection – AI Study Assistant

## Project Description
This project is a Computer Science Study Assistant designed to simulate an AI tutor. It can answer questions, summarize topics, and generate quizzes using a structured pipeline.

## What I Learned
I learned how AI systems can be structured using modular components such as intent classification, retrieval systems, and response generation. I also learned how prompts and data structure affect output quality.

## Challenges
One challenge was controlling inconsistent outputs and ensuring the system responded correctly to different types of questions.

## Solutions
I improved system reliability by simplifying the retrieval system, refining prompts, and separating logic between components.

## Future Improvements
Future versions could include a web interface, improved semantic search, and advanced transformer-based models.

## Conclusion
This project demonstrates foundational AI concepts and shows how structured systems can simulate intelligent tutoring behavior.
